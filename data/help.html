Справка

Настройки простые, есть расписание, которое управляет поливом. Из особенностей: к каждому насосу можно привязать датчик влажности грунта, можно один датчик привязать к разным насосам, или использовать среднее по всем датчикам. Но нельзя использовать несколько датчиков к одному насосу в одном расписании.

MQTT нет, управление идёт через Web или Telegram. Что-бы не разводить учётные записи можно одно устройство назначить главным (Hub), а остальные будут клиентам. И управлять всеми устройствами через одну учётную запись.

Перед тем, как использовать telegram, надо прописать параметры бота в настройках и указать список чатов, на которые он будет отвечать. Узнать номер чата можно открыв чат с ботом и набрав /chatid. Прописать настройки бота надо только на одном устройстве, иначе они будут "красть" друг у друга сообщения.

Для запуска полива надо отправить 1*N (с телефона в тональном режиме 1*N#), где N номер насоса или 0 для всех сразу.

Для изменения канала отправки уведомлений надо отправить 8*N (с телефона в тональном режиме 8*N#), где N:

  • 0 - Ничего не отправлять
  • 1 - Отправлять через Hub
  • 2 - Отправлять через Telegram/WiFi
  • 3 - Отправлять через Telegram/GPRS
  • 4 - Отправлять через SMS

Условные обозначения:

  • p - pump - насос
  • s - sensor - датчик влажности грунта
  • c - count - количество запусков
  • v - volume - объём воды

Для изменнения расписания надо прописать команду вида:
e1 7:00 (1:30) p1 p2 p3 if s1 < 50% on
где e1 - событие 1 (первая строка расписаня)
время начала и в скобках период повтора
дальше список насосов которые надо включить или pall чтобы включить все
дальше условие, номер датчика, больше или меньше и проценты влажности
on/off включить/выключить эту запись расписания.
Все парасетры обязательно должны быть разделены пробелами.

При изменении расписания строка обязательно должна начинаться с eN, порядок остальных параметров не имеет значения. Если какой-то параметр не надо менять, можно его просто пропустить. Например:
e1 off
просто выключит этот пункт расписания не затронув другие настройки.
Для отключения условия надо казать '=' или 'noif'.
Для отлючения всех насосов просто 'p'
Для включения условия по среднему значению всех датчиков 's0'
Один параметр может повторяться несколько раз, имеет значения только последний. Если указать
e1 s1 < 50% s2 > 20%
то работать будет именно датчик s2 если влажность выше 20%.
А если в конце указать 'noif', то вообще все условия выключаться.

Для работы с подчинёнными устройствами в чате Hub надо перед командой добавлять номер зарегистрированного устройства. Что-бы не делать это постоянно, можно отдать команду 'pin N' или '/N', где N номер. Отменить привязку: 'unpin' или '/'
